# dev-assist

### Usage

in `settings.gradle.kts`

```gradle
	dependencyResolutionManagement {
		//..
		repositories {
			//..
			maven { url = uri("https://jitpack.io") }
		}
	}
```

and in `build.gradle.kts`

```gradle
	dependencies {
	        implementation 'com.github.EsmaeelNabil:dev-assist:v0.0.1'
	}
```
